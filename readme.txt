== Description ==

This is plugin for Wordpress . The plugin allows to publish Certificates Wall - the set of images corresponding to certificates, awards, diplomas ect. 
This plugin works like widget. It pulls data from certificateswall.com and displays on Wordpress site.

This plugin will be useful when Wordpress site is used as personal blog. In this case blogger can represent his Certificates Wall.

The plugin has couple of configuration settings to choose best representation of Certificates Wall.

== Installation ==

1. Upload 'widget_certificateswall.php' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Or use plugin installer in wordpress sidebar and then acticate the plugin.

== Frequently Asked Questions ==

= What is Certificates Wall? =

The CertificatesWall.com provides Web service functionality for creating and sharing virtual wall with personal certificates, awards, diplomas, licences. The wall can be shared with social networks and popular web services.
